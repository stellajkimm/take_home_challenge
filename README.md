ATHLETES-R-US
===================
A simple Django CRUD app with a list of professional athletes.

INSTALLATION
------------
<pre><code>git clone <this repo>
cd take-home-challenge/
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
</code></pre>
